Stack Implementation

LIFO Structure

Structure Definition (C)

A Stack is a linear data structure following the Last-In, First-Out principle. Below is the standard array-based implementation.

#define MAX_SIZE 100
typedef struct {
  int arr[MAX_SIZE];

Singly Linked List

Dynamic Memory

Node Structure (C)

Unlike arrays, elements are not stored in contiguous memory locations. Each node links to the next.

struct Node {
  int data;
  struct Node* next;
};

Key Characteristics

  • Dynamic Size: Allocates memory as needed (heap).
  • Cache Unfriendly: Nodes are scattered in memory.
  • No Random Access: Accessing the Nth element requires O(n) traversal.

Demo: Stack Program

Complete C Implementation

Program Overview

This code implements a complete Stack using an Array.

  • push(x): Adds 'x' to top. Checks for Overflow.
  • pop(): Returns top element. Checks for Underflow.
  • peek(): View top element without removing.
  • main(): Driver function to test logic.
stack_demo.c 
// Complete C Program to Implement Stack Operations
#include <stdio.h>
#include <stdlib.h>
#define MAX 100

int stack[MAX];
int top = -1;

void push(int value) {
    if (top == MAX - 1) {
        printf("Stack Overflow\n");
    } else {
        stack[++top] = value;
        printf("Pushed %d\n", value);
    }
}

int pop() {
    if (top == -1) {
        printf("Stack Underflow\n");
        return -1;
    } else {
        return stack[top--];
    }
}

int peek() {
    if (top == -1) {
        printf("Stack is Empty\n");
        return -1;
    }
    return stack[top];
}

int main() {
    push(10);
    push(20);
    push(30);
    
    printf("Top element: %d\n", peek());
    
    printf("Popped element: %d\n", pop());
    printf("Popped element: %d\n", pop());
    
    return 0;
}

Demo: Linked List Program

Singly Linked List in C

Program Overview

Dynamic memory management using pointers.

  • struct Node: Self-referential structure.
  • insert(): Allocates heap memory (malloc).
  • delete(): Frees memory to prevent leaks.
  • display(): Traverses until NULL.
linkedlist_demo.c 
// C Program for Singly Linked List
#include <stdio.h>
#include <stdlib.h>

struct Node {
    int data;
    struct Node* next;
};

void insert(struct Node** head, int val) {
    struct Node* newNode = (struct Node*)malloc(sizeof(struct Node));
    newNode->data = val;
    newNode->next = *head;
    *head = newNode;
    printf("Inserted %d\n", val);
}

void delete(struct Node** head) {
    if (*head == NULL) return;
    struct Node* temp = *head;
    *head = (*head)->next;
    printf("Deleted %d\n", temp->data);
    free(temp);
}

void display(struct Node* node) {
    while (node != NULL) {
        printf("%d -> ", node->data);
        node = node->next;
    }
    printf("NULL\n");
}

int main() {
    struct Node* head = NULL;
    
    insert(&head, 10);
    insert(&head, 20);
    insert(&head, 30);
    
    printf("List: ");
    display(head);
    
    delete(&head);
    printf("After Deletion: ");
    display(head);
    
    return 0;
}

Frequently Asked Questions

Viva & Interview Prep
1. When should you prefer a Linked List over an Array?
Use a Linked List when you do not know the exact number of elements in advance (Dynamic Size) or when you need frequent insertions and deletions (O(1) time). Arrays are preferred for random access (O(1) access).
2. What is Stack Overflow and when does it occur?
Stack Overflow occurs when the stack memory runs out. This typically happens in deep or infinite recursion where new stack frames are pushed without popping the previous ones.
3. Why is binary search not feasible in Singly Linked Lists?
Binary Search requires random access to the middle element (O(1)). In a Singly Linked List, accessing the middle element requires traversing from the head (O(n)), making Binary Search effectively O(n) instead of O(log n).
4. What is the advantage of a Circular Queue over a Linear Queue?
In a standard array-based Linear Queue, memory at the front becomes unusable after dequeue operations. A Circular Queue connects the rear back to the front, utilizing empty spaces and preventing "false overflow."

Circular Linked List

Definition

A linked list where all nodes are connected to form a circle. There is no NULL at the end.

  • Any node can be a starting point.
  • Useful for Queue implementations.
  • Risk of infinite loops.
